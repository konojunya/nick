# nick
nick is enhance npm.

![](https://raw.githubusercontent.com/konojunya/nick/master/public/nick.png)

## Install

```
$ go get github.com/konojunya/nick
```

## Usage

save .nick.json file from package.json

```
$ nick save
```

load from .nick.json file and install modules

```
$ nick load
```

## Require

- npm
